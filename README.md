# 🎰 パチンコ計算ツール

パチンコの仕事量と収支を自動計算するWebツールです。

## 🔗 使い方

以下のリンクをクリックするだけ：

[https://kbaddy0414-alt.github.io/リポジトリ名/](https://kbaddy0414-alt.github.io/リポジトリ名/)

## 📋 機能

- 🎯 機種データ入力（交換率、確率など）
- 📊 初当りデータ管理
- 💰 仕事量自動計算
- 📈 収支計算
- 💾 データ保存/読み込み機能

## 🎮 基本的な使い方

1. **機種データを入力**
   - 交換率、トータル確率を設定
   - 玉数データを入力

2. **初当りデータを追加**
   - 「データを追加」ボタンをクリック
   - 回転数、出玉、ラウンド、時短を入力

3. **自動計算**
   - 仕事量と収支が自動で計算されます
   - データは保存ボタンを押すと保存されます

## ❓ よくある質問

**Q: データはどこに保存されますか？**  
A: ブラウザのローカルストレージに保存されます。同じブラウザで開くとデータが残っています。

**Q: スマホでも使えますか？**  
A: はい、スマホのブラウザでも問題なく使用できます。
